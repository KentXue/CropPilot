# CropPilot系统UI界面原型设计

## 1. 设计原则

### 1.1 用户体验原则
- **简洁直观**：界面简洁明了，操作流程直观易懂
- **响应式设计**：适配不同屏幕尺寸，支持移动设备
- **一致性**：保持界面元素和交互方式的一致性
- **可访问性**：支持键盘导航，考虑视觉障碍用户需求

### 1.2 视觉设计原则
- **农业主题**：采用绿色系配色，体现农业特色
- **信息层次**：通过颜色、字体、间距建立清晰的信息层次
- **数据可视化**：重要数据通过图表、图标等方式直观展示
- **状态反馈**：及时提供操作反馈和系统状态信息

## 2. 色彩系统设计

### 2.1 主色调
```css
/* 主色调 - 农业绿 */
--primary-color: #2E7D32;      /* 深绿色 */
--primary-light: #4CAF50;      /* 浅绿色 */
--primary-dark: #1B5E20;       /* 深绿色 */

/* 辅助色 */
--secondary-color: #FF9800;     /* 橙色 - 警告/提醒 */
--accent-color: #2196F3;        /* 蓝色 - 链接/按钮 */
--success-color: #4CAF50;       /* 成功状态 */
--warning-color: #FF9800;       /* 警告状态 */

/* 中性色 */
--text-primary: #212121;        /* 主要文字 */
--text-secondary: #757575;      /* 次要文字 */
--background-color: #FAFAFA;    /* 背景色 */
--surface-color: #FFFFFF;       /* 卡片背景 */
--border-color: #E0E0E0;        /* 边框颜色 */
```

### 2.2 语义化颜色
- **健康状态**：绿色系 (#4CAF50)
- **警告状态**：橙色系 (#FF9800)
- **危险状态**：红色系 (#F44336)
- **信息提示**：蓝色系 (#2196F3)
- **中性状态**：灰色系 (#9E9E9E)

## 7. 交互设计规范

### 7.1 导航设计
- **面包屑导航**：显示当前页面位置
- **标签页导航**：相关功能页面切换
- **侧边栏导航**：主要功能模块导航
- **底部导航**：移动端主要功能快速访问

### 7.2 反馈设计
- **加载状态**：显示加载进度和状态
- **成功反馈**：操作成功的确认信息
- **错误提示**：清晰的错误信息和解决建议
- **警告提醒**：重要操作的确认提示

### 7.3 数据展示
- **表格设计**：清晰的表头、行间距、对齐方式
- **图表设计**：合适的图表类型、颜色、标签
- **卡片设计**：统一的卡片样式、间距、阴影
